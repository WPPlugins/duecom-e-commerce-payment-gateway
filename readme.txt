== Description ==

Be sure to checkout our <a href="https://due.com/blog/request-access-use-due-payment-gateway-woocommerce/">tutorial</a> on using this plugin.

This plugin adds the Due.com E-Commerce Payment Gateway Support to WooCommerce for accepting **Credit Cards Payments** directly on checkout page for your WooCommerce products and subscriptions.
This plugin uses Due API **v1** to charge credit cards. For better visualization of how it looks & works check screenshots tab.

= Features =
1. Accept Credit Card Payments.
2. Supports Products and Subscriptions.
3. Give your customers the option to store and manage credit cards for future payments.
4. Supports Sandbox and Live Integration.
5. Need custom checkout experience? Contact <a href="https://support.due.com" target="_blank">Due Support</a>.
6. This plugin requires SSL.
7. This plugin supports refunds **(Only in Cents)** in woocommerce interface
8. Contribute to future releases by sending us <a href="mailto:chalmers@due.com?subject=Feature%20Request">Feature Requests</a>!

== Installation ==

1. Upload 'duecom-e-commerce-payment-gateway' folder to the '/wp-content/plugins/' directory
2. Activate 'Due.com E-Commerce Payment Gateway' from wp plugin lists in admin area
3. Plugin will appear in settings of WooCommerce
4. You can set the addon settings from 
   wocommmerce -> settings -> Checkout -> Due.com Payments
5. **Make sure you have 2 Decimal Places in Pricing Options (WooCommerce > Settings > General > Currency Options) with Trailing Zeros Checked else the charge created by card will be incorrect**

== Frequently Asked Questions ==

Be sure to checkout our <a href="https://due.com/blog/request-access-use-due-payment-gateway-woocommerce/">tutorial</a> on using this plugin.
1. You need to have WooCoommerce plugin installed to make this plugin work
2. You need to obtain you Sandbox API key and App Id from your Sandbox Due Account: https://stage-app.due.com/settings
3. You need to obtain you Live API key and App Id from your Live Due Account: https://app.due.com/settings
4. This plugin requires SSL.
5. This plugin requires CURL
6. This plugin does not support Pre Order or Subscriptions a this time. Send us a <a href="mailto:chalmers@due.com?subject=Feature%20Request">Feature Request</a> if you are interested.
7. Due & PCI compliance requires to use SSL always
8. This plugin Supports refunds **(Only in Cents)** in WooCommerce interface.
9. Contact our <a href="mailto:support@due.com?subject=WooCommerce%20Gateway%20Question">Customer Support</a> if you have any additional questions.
